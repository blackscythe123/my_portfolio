## 🔧 Setup Instructions

1. **Clone the repo**  
```bash
git clone https://github.com/blackscythe123/portfolio.git
cd portfolio
````

2. **Install dependencies**

```bash
npm install  # or yarn install
```

3. **Start the development server**

```bash
npm run dev
```

4. **Build for production**

```bash
npm run build
```
